Backpropagation in 4 Visual Steps

1. Forward

The input goes forward through the network. The neuron computes z = wx + b, then uses an activation function to produce output a.

2. Loss

The prediction is compared with the true answer. This difference becomes the loss. Smaller loss means better prediction.

3. Backward

The error moves backward. Using the chain rule, the model finds how much the weight and bias caused the error.

4. Update

The weight is changed using gradient descent: w = w - η·∂L/∂w. This makes the next prediction better.

Memory Tricks

Super-short trick: Forward → Loss → Backward → Update

Easy meaning: First the network answers, then it checks the mistake, then it blames each part, then it fixes itself.

Another analogy: Throw a ball, miss the basket, see the error, then correct your hand movement next time. That correction step is like backpropagation.
